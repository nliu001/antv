功能对比

| 功能点                      | logicflow                         | antv                                | 可否替代   |
| --------------------------- | --------------------------------- | ----------------------------------- | ---------- |
| **自定义节点**              | ✅ HTML 节点                       | ✅ 支持多种节点类型                  | ✅ 完全兼容 |
| **节点分组/嵌套**           | ✅ Group 插件                      | ✅ 内置 Parent-Child                 | ✅ 完全兼容 |
| **拖拽创建节点**            | ✅ Dnd 插件                        | ✅ 已实现                            | ✅ 完全兼容 |
| **自动布局**                | ✅ 自定义布局算法                  | ⚠️ 需自行实现                        | ⚠️ 需要开发 |
| **自动扩容**                | ❌ 不支持                          | ✅ 已实现                            | ✅ 完全兼容 |
| **节点出组检测**            | ✅支持                             | ⚠️ 需自行实现                        | ⚠️ 需要开发 |
| **画布快照**                | ✅ Snapshot 插件                   | ⚠️ 需自行实现                        | ⚠️ 需要开发 |
| **键盘快捷键**              | ✅ 部分支持                        | ✅ Keyboard 插件                     | ✅ 完全兼容 |
| **只读模式**                | ✅ isSilentMode                    | ✅ 支持                              | ✅ 完全兼容 |
| **节点禁用拖拽**            | ✅`nodeModel.draggable = false`    | ✅`node.setProp('draggable', false)` | ✅ 可替代   |
| **动态切换权限**            | ✅`lf.updateEditConfig()`          | ⚠️需手动控制                         | ⚠️ 需要开发 |
| **节点点击**                | ✅ `node:dbclick` 事件             | ✅ `node:click` / `node:dblclick`    | ⚠️ 需要开发 |
| **节点右键删除**            | ✅ `node:contextmenu` + 自定义菜单 | ✅ `node:contextmenu` + 自定义菜单   | ⚠️ 需要开发 |
| **图片创建节点并切换**      | ✅ 通过 properties.img 切换        | ✅ 通过 attrs:image/xlink:href 切换  | ⚠️ 需要开发 |
| **删除节点（级联/非级联）** | ✅ 自定义实现                      | ⚠️ 需自行实现递归删除                | ⚠️ 需要开发 |
| **保存/加载画布数据**       | ✅ `getGraphData()` / `render()`   | ✅ `toJSON()` / `fromJSON()`         | ✅ 完全兼容 |
| **连续创建节点**            | ✅ `dnd.startDrag()` 保持选中状态  | ⚠️ 需自行实现拖拽状态保持            | ⚠️ 需要开发 |
| **节点自定义数据存储**      | ✅ `properties` 对象               | ✅ `data` 属性                       | ✅ 完全兼容 |
